# Phishing-Detection-System

A machine-learning based system for detecting phishing URLs and websites.

## Getting Started

### Prerequisites

- Python 3.7 or higher
- `pip` package manager

### Setting Up a Virtual Environment

It is recommended to use a virtual environment to manage project dependencies and avoid conflicts with other Python projects.

1. **Create the virtual environment:**

   ```bash
   python -m venv venv
   ```

2. **Activate the virtual environment:**

   - On **Linux / macOS:**
     ```bash
     source venv/bin/activate
     ```
   - On **Windows (Command Prompt):**
     ```cmd
     venv\Scripts\activate.bat
     ```
   - On **Windows (PowerShell):**
     ```powershell
     venv\Scripts\Activate.ps1
     ```

   Once activated, your terminal prompt will be prefixed with `(venv)` to indicate the virtual environment is active.

3. **Install project dependencies:**

   ```bash
   pip install -r requirements.txt
   ```

### Deactivating the Virtual Environment

When you are done working on the project, you can deactivate the virtual environment by running:

```bash
deactivate
```

This command works on **all platforms** (Linux, macOS, and Windows). After running it, your terminal prompt will return to normal, indicating that you are no longer inside the virtual environment.
