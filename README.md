# Phishing-Detection-System

A machine-learning based system to detect phishing URLs and websites.

## Setup

### Prerequisites

- Python 3.7 or higher
- `pip` package manager

### Create a Virtual Environment

It is recommended to use a Python virtual environment to isolate project dependencies.

```bash
# Create a virtual environment named 'venv'
python -m venv venv
```

### Activate the Virtual Environment

**Linux / macOS:**

```bash
source venv/bin/activate
```

**Windows (Command Prompt):**

```cmd
venv\Scripts\activate.bat
```

**Windows (PowerShell):**

```powershell
venv\Scripts\Activate.ps1
```

Once activated, your terminal prompt will be prefixed with `(venv)` to indicate the virtual environment is active.

### Install Dependencies

```bash
pip install -r requirements.txt
```

### Deactivate the Virtual Environment

When you are done working on the project, you can deactivate the virtual environment by running:

```bash
deactivate
```

This command works on **all platforms** (Linux, macOS, and Windows). After running it, your terminal prompt will return to normal and you will be back to using the system-wide Python installation.
